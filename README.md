# FocusForge ⚔️

A productivity RPG where tasks are quests and focus sessions are battles.

## Live Demo
https://riyadhami13.github.io/FocusForge/

## How to Play
1. Go to Quest Log and add tasks as quests
2. Target a quest to battle it
3. Start a 25 min focus session on The Forge
4. Complete sessions to earn XP
5. Complete quests for bonus XP
6. Come back daily to keep your streak alive

## Tech Stack
- HTML
- CSS
- JavaScript
- No frameworks, no libraries

## Features
- Pomodoro timer with focus and rest modes
- XP and leveling system
- Daily streak tracker with multiplier
- Quest system with 4 difficulty levels
- All data saved locally in your browser
