# FocusForge ⚔️

A productivity RPG where tasks are quests and the focus 
sessions are battles.

## Live Demo
https://riyadhami13.github.io/FocusForge/

## Features
- ⏱ Pomodoro timer (25 min focus / 5 min rest)
- ⚔️ Quest system with 4 difficulty tiers
- ⚡ XP and leveling system
- 🔥 Daily streak tracker with multiplier (up to 2.5x)
- 🔴 Boss Battle Mode — red pulsing ring
- 🔊 Sound effects for every action
- ✅ Completed quests section
- 📛 Custom player name
- 💾 All data saved in localStorage

## Tech Stack
HTML, CSS, JavaScript — zero frameworks, zero libraries

## How to Play
1. Add quests in Quest Log
2. Target a quest
3. Start a focus session on The Forge
4. Complete sessions and quests to earn XP
5. Come back daily to keep your streak alive
